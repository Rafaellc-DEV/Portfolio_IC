Projeto 3: Carro de controle remoto com Arduino (bluetooth)

Este projeto foi desenvolvido para controlar um carro de controle remoto utilizando um Arduino. A comunicação entre o Arduino e o celular é feita através de um módulo Bluetooth, permitindo o pareamento com o aplicativo Bluetooth RC Controller. O objetivo inicial foi criar um sistema básico de controle remoto, com a intenção de, no futuro, implementar funcionalidades baseadas em machine learning, trazendo mais inteligência e automação para o controle do carro.
